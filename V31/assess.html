Is What We're Doing Actually Working?

When a community puts effort into something, this tool helps see whether it's making a difference. The answers go back to the community, not up to an office.

How this works — the four-question cycle

This tool runs on a cycle. Each time you assess a program, you're answering one of four questions:

1. Plan — What was this supposed to accomplish?
2. Measure — What's actually happening? (Traffic lights make this visible.)
3. Improve — What should change? (Your evidence notes capture this.)
4. Verify — Did it change? (Come back in 3 months and assess again. Compare.)

Green = working. Yellow = needs attention. Red = action needed. The same program assessed twice shows whether anything changed.
